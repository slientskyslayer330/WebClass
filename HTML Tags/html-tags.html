P tag is for paragraphs. Within a paragraph, you can bold the words either with b (bold) tag or strong tag, italic the words either with italic tag or em (emphasize) tag . Wirte a superscript with sup tag - 8th and a subscript with sub tag - H2O.

NOTE! u tag is used to underline the word but it is deprecated in HTML 5. You can view a list of deprecated html tags and attributes on Turoial Points. And please avoid using them. We have CSS for text decoration.


NOTE! html neglects white spaces (i.e. line breaks, multiple spaces or tabs). Thus, use br tag for line break, and " " for non-breaking space (i.e if u add " " for five times, it will add a non-breaking space equal to 5 spaces.
